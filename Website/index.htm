Summary

Drone based mixed reality is a system to bridge the phyiscal and digital worlds using drones. We will be turning any surface into a digital surface with an interface. A drone will follow the user around the room and a pico projector will project an appropriate interface onto the object. For example, imagine walking to the worktable and projecting a circuit simulator on the workbench itself. Walk to the coffeetable and make a power point presentation on the table to a group standing around. Head to the speaker system and an interface pops up on the wall next to it. You increase the volume at the flick of your hand gesture and change the song without once pulling out your laptop or phone. With tangible user interfaces we can further blur the line between physical and digital.

Fabrication

The drone will be carrying a pico projector as well as a microcontroller. A custom housing encasing will be fabricated using 3d-printing and laser cutting. This encasing will be designed to be light weight making the drone more efficient, yet strong enough to withstand drops and crashes.

Custom Electronics

The pico projector used to proejct the interfaces will be interfacing with a beaglebone microcontroller. This system will involve custom software to provide the digital output. There will be custom power electronics for regulation and recharging.

3D Tracking

For this prototype the OptiTrack camera system is used for tracking. This requires extensive setup and space. In a final version, an inside out tracking system using optical flow and camera systems would be used. The user can interact with hand gestures tracked through a Leap Motion.

Custom Wireless Communication

Custom wireless communication will be implemented between the BeagleBone microcontroller for the projector and the pc processing the tracking and user inputs.

Output

The output of the system is an interface that can be changed and modified per object projected on. This mobile interface allows users to more easily and intuitively interact with their world.

Applications

We see several applications for this drone based MR system. Users can interact with IoT parts of their home. Turn up the heat, change all of the lights, lock the doors, etc. Users can present to small groups easily. This can also enable spatial Augmented Reality.
